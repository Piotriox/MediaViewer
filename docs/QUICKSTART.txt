# MediaViewer - Quick Reference

## Installation

```bash
pip install -r requirements.txt
```

## Running

```bash
# Basic launch
python main.py

# With image
python main.py "image.jpg"

# With video
python main.py "video.mp4"
```

## Image Controls

- **Scroll Up**: Zoom in
- **Scroll Down**: Zoom out
- **Drag**: Pan around image

## Video Controls

- **Play/Pause**: Click button
- **Timeline**: Drag to seek
- **Volume**: Adjust slider

## Supported Formats

**Images**: JPG, PNG, GIF, BMP, WebP, TIFF, ICO

**Videos**: MP4, MKV, AVI, MOV, FLV, WMV, WebM, M4V, 3GP

## Package Application

```bash
pip install pyinstaller
pyinstaller --onefile --windowed main.py
# Result: dist/main.exe
```
